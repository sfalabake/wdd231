Tech Tools Directory - Site Plan

A planning document for the WDD231 final project

Site Name

Tech Tools Directory

The name was selected because it clearly communicates the purpose of the website: a structured directory of useful technology tools for students, developers, and professionals.

Site Purpose

The Tech Tools Directory provides users with a curated collection of technology tools across categories such as programming, design, artificial intelligence, and productivity. Users can browse tools, search for specific solutions, view detailed tool information, and submit suggestions for new tools.

Scenarios

  • What AI tools can help me improve productivity and learning efficiency?
  • Where can I find free tools for web development and coding practice?
  • Which design tools are best for UI/UX projects and graphic design?
  • How can I recommend a useful tech tool to be added to the directory?

Color Scheme

#003366 - Primary Blue

Used for headers, navigation, and footer background.

#00A6A6 - Teal

Used for buttons, links, and hover effects.

#F4F4F4 - Light Gray

Used for page background and content sections.

#333333 - Dark Gray

Used for body text and general content.

Typography

Poppins: Used for headings and navigation.

Roboto: Used for body text and descriptions.

Montserrat: Used for buttons and highlights.
